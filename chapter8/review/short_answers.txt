1. What does the following code display?
    mystr = 'abc'
    mystr2 = '123'
    mystr += mystr2
    print(mystr)

Sol: abc123

2. What does the following code display?
    mystr = 'abc' * 3
    print(mystr)

Sol: abcabcabc

3. What will the following code display?
    mystr = 'abracadabra'
